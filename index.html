Academic Gender Search Analysis

This project analyzes the gender distribution among chief investigators in academic research. The interactive visualizer provides insights into researcher demographics, affiliations, and research areas.

📊 Dataset Overview

1,531 researchers analyzed with comprehensive gender analysis and research area categorization.

🔍 Search & Filter

Search across names, affiliations, research areas, and summaries with real-time filtering.

📈 Statistics Dashboard

View gender distribution and confidence levels with interactive charts.

🎨 Clean Design

Modern, responsive interface with color-coded badges and expandable content.

📱 Mobile Friendly

Fully responsive design that works on desktop, tablet, and mobile devices.
